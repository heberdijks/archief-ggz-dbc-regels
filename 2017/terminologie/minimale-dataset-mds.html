Minimale dataset (mds)

Via de minimale dataset leveren zorgaanbieders gegevens uit de basisregistratie aan, die inzicht geven in de geleverde en gedeclareerde zorg. In de mds staan zowel data die verplicht moeten worden aangeleverd als ook data die optioneel kunnen worden aangeleverd. De mds wordt vastgesteld door de NZa.
